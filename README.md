# Angry Birdish

A lightweight, browser-based tribute to Angry Birds built with HTML canvas and vanilla JavaScript. Launch the red bird from a slingshot, topple blocks, and clear the green pigs.

## Play locally
1. Open `index.html` in your browser (no build step required).
2. Drag the bird away from the slingshot post to aim, then release to launch.
3. Hit pigs or knock down blocks to earn points. Press **R** or click **Reset** to start a new round. The HUD shows shots fired, pigs remaining, and status messages.

## Controls
- **Drag** the bird: Aim and set power.
- **Release**: Fire the bird.
- **R** or **Reset button**: Reload the level.
